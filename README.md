# Binance Academy Trading Bot

An automated trading bot for Binance Academy that creates accounts, executes trades, and monitors profits.

## Features
- Automatic account creation
- Continuous trading execution
- Profit monitoring and withdrawal
- Headless browser automation
- GitHub Actions integration

## Requirements
- Verified Binance Academy account
- GitHub account with workflow permissions
- Python 3.10+
